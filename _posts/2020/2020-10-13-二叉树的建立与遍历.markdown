## 先导知识

### 树的相关知识点：
1. 结点的度：结点拥有的子树个数
2. 叶子：度数为0的结点
3. 孩子：结点子树的根
4. 双亲：孩子结点的上层结点
5. 结点层次：根节点算起，根为第一层，他的孩子为第二层
6. 树的深度：树的层数
7. 完全二叉树：深度为k，有n个结点的二叉树当且仅当其每一个结点都与深度为k的满二叉树中编号从1到n的结点一一对应时，称为完全二叉树
8. 满二叉树：如果一棵二叉树只有度为0的结点和度为2的结点，并且度为0的结点在同一层上，则这棵二叉树为满二叉树。

### 二叉树的性质：
1. 二叉树的第i层上至多有2^(i-1)（i≥1）个节点
2. 深度为h的二叉树中至多含有2h-1个节点
3. 若在任意一棵二叉树中，有n0个叶子节点，有n2个度为2的节点，则必有n0=n2+1
4. 具有n个节点的完全二叉树深为log2x+1（其中x表示不大于n的最大整数）
5. 对于编号为i（i≥1）的节点：当i=1时，该节点为根，它无双亲节点
6. 对于编号为i（i≥1）的节点：当i>1时，该节点的双亲节点的编号为i/2
## 二叉树的建立
二叉树的建立有多种方法，例如数组建立或者链式存储的方法建立，这里我们用最常规的链式存储建立一颗二叉树。
一般题目中都是给一棵树的先序遍历。如：
**$~$A$~$B$~$C$~$\*$~$\*$~$\*$~$D$~$\*$~$E$~$\*$~$\***
一般星号代表空节点，先序遍历的遍历过程是根左右，这对于这可树的任何一个子树都是成立的。所以这可树画出来就长这个亚子:

### 结点的保存

树每一个的结点都有一个数据和两个分别指向左右子树的指针，显然需要一个结构体，这里我们用**typedef**来定义其指针。
```c++
typedef struct BiTNode{
    char data;
    BiTNode *lchild,*rchild;
}BiTNode,*BiTree;

````
显然，找到一个树的根，我们就能找到这一整棵树。所以我们可以在主函数中定义一个根结点，然后把它传到建树的函数中。

>注：传入函数的需要是指向指针的指针或指针的引用，否则出函数之后，这棵树就会消失。参考形参与实参。

### 建立的函数：Creat_BiTree()

在建树的过程中，每次读入一个字符，当字符为*时，当前子树为空子树，也就不用继续往下遍历了。否则，将字符给当前结点，继续遍历它的左子树，左子树遍历完再遍历右子树。显然这个过程需要通过递归来实现。所以我们先要知道如何保存树的结点。
``` c++
void Creat_BiTree(BiTree &T){
    char ch;
    cin>>ch;
    if(ch=='*')
        T=NULL;
    else{
        T=(BiTree)malloc(sizeof(BiTNode));
        T->data=ch;
        Creat_BiTree(T->lchild);
        Creat_BiTree(T->rchild);
    }
}
```

##  树的遍历 **_Traverse()
#### 树的遍历一共有4种：
>   1.  **先序遍历** 根 左 右
>   2.  **中序遍历** 左 根 右
>   3.  **后序遍历** 左 右 根
>   4.  **层序遍历** 根据树的深度，一层一层遍历

### 先序中序后序
对于一颗树来说，每一个结点都是其所在子树的根，所以它的左右孩子又是其所在左右子树的根。所以先序中序后序都可以通过递归的方式实现。只不过对于每一个结点先序是先访问根，再左子树，再右子树,遇到空子树就返回。而中序是先访问左子树，再访问根，再右子树，后序同理。所以只要在函数中调整一下语句的顺序即可得到三种遍历。
``` c++
//先序遍历
void Front_Traverse(BiTree T){
    if(T==NULL)
        return;
    cout<<T->data<<" ";
    Front_Traverse(T->lchild);
    Front_Traverse(T->rchild);
}
//中序遍历
void Mid_Traverse(BiTree T){
    if(T==NULL)
        return;
    Mid_Traverse(T->lchild);
    cout<<T->data<<" ";
    Mid_Traverse(T->rchild);
}
//后序遍历
void Back_Traverse(BiTree T){
    if(T==NULL)
        return;
    Back_Traverse(T->lchild);
    Back_Traverse(T->rchild);
    cout<<T->data<<" ";
}
```
### 层序 BFS
层序遍历比较特殊，因为不能通过指针的指向关系得到层序遍历。但是，可与发现每一次的结点都是上一层的结点的孩子。如果每一层的结点都找到，在通过这些结点找到他们的孩子结点，就相当于找到了下一层的全部结点，再把这些结点全部删除，不就到下一层了嘛。显然这是一种先进先出的过程。第一层进，找到第二层的点，第一层出，找到第三层的点，第二层出。所以这个过程可以看出是一种队列。所以我们可以用队列来实现树的层序遍历。

先将根入队，之后对于每一个队首元素，先输出，再看他的左右孩子是否为空，如果不为空则入队，然后将队首出队。往复循环直到队列为空。
``` c++
//层序遍历
void Floor_Traverse(BiTree T){
    queue<BiTree> q;
    q.push(T);
    while(!q.empty()){
        cout<<q.front()->data<<" ";
        if(q.front()->lchild!=NULL)
            q.push(q.front()->lchild);
        if(q.front()->rchild!=NULL)
            q.push(q.front()->rchild);
        q.pop();
    }
}
```
对于二叉树来说还有许多骚操作，可以说树是一种十分重要的数据结构。
