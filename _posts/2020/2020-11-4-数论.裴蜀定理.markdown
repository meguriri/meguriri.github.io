## 引入
我们以一道题为例：
链接:[cgoj_2925](http://cg.sau.edu.cn/acm/submit.jsp?problemID=2925&pageNo=2&pages=61)

>给你两个容积分别为a,b（0<a,b<=1e18）的水桶，再给你一个无限容量的水缸，要求你从村口的小河里打出精确的容积为c的水装到水缸里。（只有水桶能够用来打水。注意，水桶打水必须得到准确的体积，也就是说，你不能靠估算水的体积来打水。）你要判断这个任务可不可能完成。
**【输入形式】**
多组用例，每组输入三个数a,b,c，当输入三个0时，输入结束。
**【输出形式】**
输出“T”表示能完成，“F”表示做不到。（不包含引号）
**【样例输入】**
4 2 3
5 3 3
0 0 0
**【样例输出】**
F
T

不难分析，对于容积为$a,b$的两个水桶，假设需要容积为$a$的水桶$x$桶容积为$b$的水桶$y$桶。那么能完成的情况就是$ax+by=c$存在整数解。即：
$$ax+by=c (x,y\in N)$$

那么怎么求x，y呢，我们很容易想到暴力的做法，在$ax+by<=c$的前提下用双重循环遍历x，y直到找到解。然而本题的数据过大，用$O(n^2)$的算法肯定不能过。这时候这种形如$ax+by=c$存在整数解的问题都可以通过裴蜀定理来求解。

##裴蜀定理
>定理：对于给定的正整数a，b，方程$ax+by=c$有解的充要条件为c是$gcd(a,b)$的整数倍。
